About Me

In front-end development, I have a deep understanding of HTML, CSS, and JavaScript, which allows me to create user interfaces that are both visually appealing and highly functional. I have experience in building responsive and mobile-friendly web applications that can be accessed from any device.

In UI/UX design, I have a strong grasp of the principles of user-centered design, which enables me to create interfaces that are easy to use, intuitive, and visually appealing. I'm familiar with the latest design trends and have experience in creating wireframes, prototypes, and mockups using tools like Sketch and Adobe XD.

My Project

Conducting research on users and the product to be created. This can be done through interviews with users, competitor analysis, and observing users while using the product.

Creating a user persona, which is a description of the characteristics and needs of users portrayed in a profile.

Creating a user flow, which is a description of the path taken by users when using the product from start to finish.

Creating a wireframe, which is a layout and navigation display of the product that is still a sketch without detailed design.

Conducting testing of the product with users to get feedback and evaluation to improve product quality.
